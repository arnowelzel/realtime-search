Search results in realtime with ajax for WordPress.

== Description ==

This plugin adds realtime search using ajax to WordPress. While typing in the search box, the results will automatically be listed in a popup result list which can be navigated using the keyboard as well.

== Installation ==

1. Extract the contents of the package to the `/wp-content/plugins/realtime-search` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
